## 四、观察项（非阻断，建议后续迭代处理）

1. **rag_router / study_router 缺少 require_auth**: 这两个路由的 endpoint 未挂 `Depends(require_auth)`。经 git diff 确认是历史遗留（第二轮仅添加了 `await` 异步化修复，未移除已有保护）。建议后续迭代统一补全。

2. **CI 测试步骤使用 `|| true`**: ci.yml 中 pytest 步骤末尾有 `|| true`，意味着测试失败不会阻断 CI。这在初期阶段是合理的（避免环境问题阻断流水线），但建议稳定后移除 `|| true` 以实现真正的质量门禁。

3. **cet4-core.json 条目数 3629 vs 任务描述 3259**: 实际数据量超出预期（3629 > 3259），数据更丰富，无负面影响。
